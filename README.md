# encantar2

A next-generation GPU-accelerated Augmented Reality framework for the web.

## Overview

encantar2 is the evolution of the encantar.js AR framework, designed to provide:

- **Cross-platform compatibility**: iOS, Android, Desktop browsers
- **High performance**: GPU-accelerated processing
- **Easy integration**: Simple API for developers
- **No app required**: Runs directly in modern web browsers
